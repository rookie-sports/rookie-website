# rookie-website

This repository contains the official ROOKIE company website.

## Overview

A minimal, single-page website featuring the ROOKIE company name and tagline. The design follows a clean, terminal-inspired aesthetic with no external dependencies.

## Structure

- `index.html` - The main website file with embedded CSS

## Design Principles

- Minimal and clean design
- Terminal/plain-text aesthetic
- No JavaScript or external dependencies
- Fully responsive for mobile and desktop
- Accessible and fast-loading

## Deployment

This is a static HTML page that can be served from any web server or static hosting platform.
