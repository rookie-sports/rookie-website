# rookie-website

Official ROOKIE landing page — minimal, brutalist design inspired by The Interface.

## Overview

A minimal, single-page website featuring the ROOKIE brand and tagline. The design follows a clean, brutalist aesthetic with Times New Roman typography, white background, and black text.

## Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet with brutalist/minimal design
- `.cloudflare-pages-config.json` - Cloudflare Pages configuration
- `.github/workflows/cloudflare-pages.yml` - Automated deployment workflow

## Design Principles

- **Brutalist/minimal aesthetic** inspired by The Interface
- **Times New Roman typography** for classic, editorial feel
- **White/near-white background** (#fafafa) with black text
- **Centered layout** with maximum simplicity
- **Logo placeholder** — bordered box with ROOKIE text (owner will provide actual logo)
- **No JavaScript** — pure HTML + CSS
- **No frameworks** — zero dependencies
- **Fully responsive** for mobile and desktop

## Deployment

Automatically deploys to Cloudflare Pages (free tier) on push to main branch via GitHub Actions.

## Local Development

Simply open `index.html` in a browser. No build process required.
